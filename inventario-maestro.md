# INVENTARIO MAESTRO — El Cazador de Medianoche

## GUÍA PARA EL ESCRITOR
Este documento rastrea **todos los objetos** que los personajes adquieren, pierden, usan o comercian.
- **[PAYOFF]** = El objeto tiene un uso posterior confirmado
- **[AMBIENTE]** = El objeto existe para dar vida al mundo, sin payoff mecánico
- **[COMERCIO]** = El objeto fue comprado/intercambiado en una aldea
- **[PENDIENTE]** = El objeto necesita un payoff que aún no está escrito

## SISTEMA DE COMERCIO POR ALDEA

### Economías únicas establecidas en la historia:

| Aldea/Grupo | Moneda/Sistema | Qué valoran | Qué ofrecen |
|-------------|---------------|-------------|-------------|
| **Mapaches Cartógrafos** (sec 02) | Monedas de plata | Dinero contante, tratos rápidos | Mapas, información de rutas, "provisiones" dudosas |
| **Zorros Plateros** (sec 05) | Mixto: moneda + servicio | Trabajo honesto, habilidad manual | Herrería, flechas, reparaciones, hospitalidad |
| **Armadillos Cocineros** (sec 07B) | Participación alegre | Entusiasmo, compañía, ayuda voluntaria | Comida transformadora, mapas, alojamiento |
| **Nutrias Velocistas** (sec 08A) | Trueque de habilidades | Habilidades prácticas, buen humor | Cera impermeable, herramientas acuáticas, terapia |
| **Murciélagos Críticos** (sec 08B) | Intercambio de conocimiento | Información, experiencias gustativas | Medicinas sensoriales, pergaminos, especias |
| **Mariposas Mensajeras** (sec 08C) | Historias verdaderas | Verdad emocional, vulnerabilidad | Polvo revelador, mensajería, flores preservadas |
| **Tejones Excavadores** (sec 09A) | Prueba de cooperación | Trabajo en equipo demostrado | Herramientas anti-magia, cristales, cuerda de confianza |

### Patrón narrativo:
- Cada economía refleja los **valores de la especie**
- El "precio" sube emocionalmente: de monedas (fácil) → habilidades → conocimiento → verdad personal (difícil)
- Esto espeja el arco de Seraphina: cada vez paga con más de sí misma

## REGLAS PARA EL ESCRITOR

1. **Nunca introducir un objeto sin anotar aquí si tiene payoff o no**
2. **Mantener ratio ~50% payoff mínimo** (actualmente 80%, podemos relajarlo)
3. **Los objetos deben obtenerse de formas que revelen personaje:**
   - Axel obtiene cosas ofreciendo habilidades prácticas (afilar, rastrear)
   - Ember obtiene cosas cocinando o siendo emocionalmente generoso
   - Seraphina obtiene cosas pagando con verdad (el precio más alto para ella)
   - Eco obtiene cosas intercambiando conocimiento
4. **Cada aldea tiene una economía diferente** — nunca repetir el sistema de trueque
5. **Los payoffs deben sentirse naturales**, no forzados. Si un objeto se usa después, debe parecer que el personaje simplemente recordó que lo tenía, no que fue plantado
6. **Actualizar este documento** cada vez que se inserte un objeto nuevo en cualquier sección
